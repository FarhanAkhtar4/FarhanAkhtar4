## About Me

Research-driven ML Systems Engineer building production-grade AI systems — from transformer-based predictive models to retrieval-augmented generation pipelines. At **NIT Calicut**, I developed deep learning applications for engineering problems, specifically building a **Temporal Fusion Transformer that achieved 22% improvement** over XGBoost and KNN baselines for seismic response prediction. My work spans PyTorch model development, systematic hyperparameter tuning, and rigorous validation against baselines.

I design agentic AI workflows that combine LLM reasoning with retrieval systems, and build attention-based architectures for both time-series and tabular data. Every project follows a disciplined approach: define the problem, validate against baselines, and deliver measurable results.
